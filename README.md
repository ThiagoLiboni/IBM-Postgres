# INICIAR SERVIÇO

## Pré-Requisitos
- Install docker e docker-compose (https://docs.docker.com/desktop/setup/install/windows-install/)

## Criar Container
- Execute o seguinte comando para construir e iniciar o contêiner:
     ```bash
     docker-compose up --build
     ```
## Acessando Serviço
- (localhost:3000)
